# telcosense-webapp-frontend

Frontend web application for the TelcoSense platform.

## Overview

This repository contains the frontend of the TelcoSense web application, providing an interactive web-based user interface for data visualization, exploration, and control. The frontend integrates multiple data layers and services exposed by the TelcoSense backend and related processing components.

It is designed as a modern single-page application with a focus on performance, clarity, and extensibility.

## How to Run

1. Install **Bun**: https://bun.sh/
2. Open this project in **VS Code**
3. Run `bun install` in the project root directory
4. Run `bun dev` in the project root directory

The development server will start and provide a local URL for accessing the application.

## Contents

- Web-based user interface components
- Data visualization and interaction logic
- Integration with TelcoSense backend APIs
- Configuration and build scripts
